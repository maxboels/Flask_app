Part Two
Implement your calculator in infix notation with support for full-parenthesized operands. It's OK to assume that all the tokens are space-separated, including the parentheses tokens
Sample input (caret prompt for clarity only):
ex_1: ( 1 + 2 )
3
ex_2: ( 1 + ( 2 * 3 ) )
7
ex_3: ( ( 1 * 2 ) + 3 )
5
ex_4: ( ( ( 1 + 1 ) / 10 ) - ( 1 * 2 ) )
-1 (or -1.8)
